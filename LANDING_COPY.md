## Messaging Hierarchy

| Level | Message |
|-------|---------|
| Core value prop | Find your exact AI waste in dollars |
| Proof point | 10 minutes, no integrations needed |
| Differentiator | AI verdict that gives a number, not a hedge |
| Trust signal | Built for startups, free to start |

Every section maps back to the core value prop. No section introduces
a new idea — they all reinforce the same promise from a different angle.

## Copy Principles Used

1. **Specificity over generality** — "10 minutes" not "quick", "$X wasted" not "savings"
2. **Outcome-first** — lead with what the founder gets, not what the tool does
3. **Short sentences** — founders scan on mobile, in between meetings
4. **Mirror user language** — "waste", "cut", "sprawl" came directly from interviews
5. **No feature-speak** — "audit engine" never appears in user-facing copy
